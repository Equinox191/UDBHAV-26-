Explain binary search.
U
AI
Binary search is a way to find an item in a sorted list by repeatedly cutting your search range in half.
javascriptcopy
function binarySearch(arr, target) {
  let lo = 0, hi = arr.length - 1;
  while (lo <= hi) {
    const mid = Math.floor((lo + hi) / 2);
    if (arr[mid] === target) return mid;
    arr[mid] < target ? lo = mid + 1 : hi = mid - 1;
  }
  return -1;
}
Analogy · Think of looking up a name in a phone book. Flip to the middle, go left or right, repeat. That's O(log n).
